Hospital Management System
Project Overview

This Hospital Management System is a college-level project developed in Java to manage basic hospital operations such as patients, doctors, and appointments. The system uses a simple console-based interface to perform CRUD operations on hospital records.

Features

Add, view, and delete patient records

Add, view, and delete doctor records

Schedule and display appointments

Basic console-based user interaction

Tech Stack
Component	Technology
Language	Java
IDE	IntelliJ / Eclipse / VS Code
Build	Java Compiler (javac)
